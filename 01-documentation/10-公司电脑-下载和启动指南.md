# 🏢 公司电脑 - 下载和启动指南

> Windows 电脑专用，使用 Git Clone 方式

**⚠️ 重要说明**：本指南适用于**不能直接访问 GitHub 网页下载**的情况，使用 Git 命令行或 VS Code 的 Git 功能来获取代码。

---

## 🎯 推荐方式：使用 Git Clone

### 第一步：检查 Git 是否可用

1. **打开 CMD 或 PowerShell**
   - 按 `Win + R`，输入 `cmd` 或 `powershell`，按回车

2. **检查 Git 版本**：
   ```bash
   git --version
   ```
   
   - **如果显示版本号**（如 `git version 2.xx.x`）→ ✅ 可以使用，继续下一步
   - **如果显示"不是内部或外部命令"** → ❌ 需要安装 Git（见下方"安装 Git"部分）

### 第二步：Clone 代码

1. **进入你想保存代码的目录**
   ```bash
   # 例如：进入桌面
   cd C:\Users\你的用户名\Desktop
   
   # 或者进入文档文件夹
   cd C:\Users\你的用户名\Documents
   ```

2. **Clone 仓库**
   ```bash
   git clone https://github.com/huhaheng/riskgen-ai-demo.git
   ```

3. **等待下载完成**
   - 会显示类似：`Cloning into 'riskgen-ai-demo'...`
   - 完成后会显示：`done.`

4. **进入项目文件夹**
   ```bash
   cd riskgen-ai-demo
   ```

### 第三步：用 VS Code 打开

1. **打开 VS Code**

2. **File → Open Folder**（文件 → 打开文件夹）

3. **选择 `riskgen-ai-demo` 文件夹**

4. **点击"选择文件夹"**

### 第四步：启动项目

#### 🌟 方法 A：使用 Live Server（最推荐）

1. **安装 Live Server 扩展**（一次性操作）
   - 点击 VS Code 左侧的扩展图标（四个方块）
   - 搜索：`Live Server`
   - 找到作者是 `Ritwick Dey` 的那个
   - 点击"安装"

2. **启动项目**
   - 在 VS Code 中找到 `index.html` 文件
   - **右键点击** `index.html`
   - 选择 **"Open with Live Server"**

3. **自动打开浏览器**
   - 地址通常是：`http://127.0.0.1:5500`
   - 页面自动加载

4. ✅ **完成！**

#### 💻 方法 B：使用 Python 命令行

1. **在 VS Code 中打开终端**
   - 菜单：Terminal → New Terminal
   - 或快捷键：`` Ctrl+` ``（左上角的反引号键）

2. **输入命令**：
   ```bash
   python -m http.server 8000
   ```
   
   如果不行，试试：
   ```bash
   python3 -m http.server 8000
   ```

3. **打开浏览器**，访问：
   ```
   http://localhost:8000
   ```

4. ✅ **完成！**

## 🔄 将来如何更新代码

如果代码有更新，在公司电脑上获取最新版本：

### 方法 1：使用 Git Pull（推荐）

1. **打开 CMD 或 PowerShell**

2. **进入项目目录**
   ```bash
   cd C:\Users\你的用户名\Desktop\riskgen-ai-demo
   ```

3. **拉取最新代码**
   ```bash
   git pull
   ```

4. ✅ **完成！**代码已更新

### 方法 2：使用 VS Code 的 Git 功能

1. **在 VS Code 中打开项目**

2. **点击左侧的源代码管理图标**（分支图标）

3. **点击"..."菜单** → **"拉取"**（Pull）

4. ✅ **完成！**

## 🔐 Git 认证（如果需要）

如果 Clone 时提示需要认证：

### 方式 1：使用 Personal Access Token

1. **在能访问 GitHub 的电脑上生成 Token**
   - GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
   - Generate new token (classic)
   - 勾选 `repo` 权限
   - 复制生成的 Token（只显示一次！）

2. **Clone 时使用 Token**
   ```bash
   git clone https://你的Token@github.com/huhaheng/riskgen-ai-demo.git
   ```

### 方式 2：使用 SSH（如果公司允许）

1. **生成 SSH Key**（在能访问 GitHub 的电脑上）
2. **添加到 GitHub**
3. **使用 SSH 地址 Clone**
   ```bash
   git clone git@github.com:huhaheng/riskgen-ai-demo.git
   ```

### 第五步：测试功能

1. **点击 "New Project"** → 创建项目
2. **点击 "Knowledge Base"** → 查看 4 个案例
3. **进入项目 → Risk Report** → 查看风险数据

## ⚠️ 常见问题

### Q1: Git Clone 提示需要认证？
**A**: 几种解决方案：
1. **使用 Personal Access Token**（见上方"Git 认证"部分）
2. **使用 SSH**（如果公司允许）
3. **联系 IT 部门**申请访问权限

### Q2: Git 命令不识别？
**A**: 
- 需要安装 Git（见上方"安装 Git"部分）
- 或联系 IT 部门安装

### Q3: Clone 失败，提示连接超时？
**A**: 
- 检查网络连接
- 可能需要配置代理（联系 IT 部门）
- 或使用公司内网的 Git 服务器（如果有）

### Q4: Live Server 提示端口被占用？
**A**: 换个端口：
1. VS Code 设置（Ctrl+,）
2. 搜索：`liveserver.settings.port`
3. 改成：`5501` 或 `8080`

### Q5: Python 命令不识别？
**A**: 可能没有安装 Python，用 Live Server 方式（方法 A）

### Q6: VS Code 不能安装扩展？
**A**: 公司限制，用方法 B 或申请权限

### Q7: 页面打开是空白的？
**A**: 
- 确保用了本地服务器（不要直接双击 index.html）
- 按 F12 检查控制台错误
- 强制刷新：Ctrl+Shift+R
