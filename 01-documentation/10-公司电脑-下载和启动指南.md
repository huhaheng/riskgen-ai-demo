# 🏢 公司电脑 - 下载和启动指南

> Windows 电脑专用，超级简单的操作步骤

---

## 🎯 最简单的方式（推荐）

### 第一步：下载代码

1. **打开浏览器**，访问：
   ```
   https://github.com/huhaheng/riskgen-ai-demo
   ```

2. **点击绿色的 "Code" 按钮**
   
3. **点击 "Download ZIP"**

4. **等待下载完成**
   - 文件名：`riskgen-ai-demo-main.zip`
   - 保存位置：通常在"下载"文件夹

### 第二步：解压文件

1. **找到下载的 ZIP 文件**
   - 通常在：`C:\Users\你的用户名\Downloads\`

2. **右键点击 ZIP 文件**

3. **选择"解压到当前文件夹"**或"解压到 riskgen-ai-demo-main\"

4. **完成！**得到 `riskgen-ai-demo-main` 文件夹

### 第三步：用 VS Code 打开

1. **打开 VS Code**

2. **File → Open Folder**（文件 → 打开文件夹）

3. **选择刚才解压的文件夹**
   - `riskgen-ai-demo-main`

4. **点击"选择文件夹"**

### 第四步：启动项目

#### 🌟 方法 A：使用 Live Server（最推荐）

1. **安装 Live Server 扩展**（一次性操作）
   - 点击 VS Code 左侧的扩展图标（四个方块）
   - 搜索：`Live Server`
   - 找到作者是 `Ritwick Dey` 的那个
   - 点击"安装"

2. **启动项目**
   - 在 VS Code 中找到 `index.html` 文件
   - **右键点击** `index.html`
   - 选择 **"Open with Live Server"**

3. **自动打开浏览器**
   - 地址通常是：`http://127.0.0.1:5500`
   - 页面自动加载

4. ✅ **完成！**

#### 💻 方法 B：使用 Python 命令行

1. **在 VS Code 中打开终端**
   - 菜单：Terminal → New Terminal
   - 或快捷键：`` Ctrl+` ``（左上角的反引号键）

2. **输入命令**：
   ```bash
   python -m http.server 8000
   ```
   
   如果不行，试试：
   ```bash
   python3 -m http.server 8000
   ```

3. **打开浏览器**，访问：
   ```
   http://localhost:8000
   ```

4. ✅ **完成！**

### 第五步：测试功能

1. **点击 "New Project"** → 创建项目
2. **点击 "Knowledge Base"** → 查看 4 个案例
3. **进入项目 → Risk Report** → 查看风险数据

## ⚠️ 常见问题

### Q1: 下载的文件夹名字后面有 "-main"？
**A**: 正常的！GitHub 自动添加的。
- 文件夹名：`riskgen-ai-demo-main`
- 直接使用就好，不影响功能

### Q2: Live Server 提示端口被占用？
**A**: 换个端口：
1. VS Code 设置（Ctrl+,）
2. 搜索：`liveserver.settings.port`
3. 改成：`5501` 或 `8080`

### Q3: Python 命令不识别？
**A**: 可能没有安装 Python，用 Live Server 方式（方法 A）

### Q4: VS Code 不能安装扩展？
**A**: 公司限制，用方法 B 或申请权限

### Q5: 页面打开是空白的？
**A**: 
- 确保用了本地服务器（不要直接双击 index.html）
- 按 F12 检查控制台错误
- 强制刷新：Ctrl+Shift+R
